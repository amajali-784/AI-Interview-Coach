# 🤖 AI Interview Coach

An AI-powered interview preparation platform that simulates real-world technical and HR interviews using Large Language Models (LLMs). The application evaluates candidate responses, provides personalized feedback, analyzes performance, and helps users improve their interview skills through interactive mock interview sessions.

---

## 🚀 Features

* 🎯 AI-generated interview questions based on job role and experience
* 💬 Technical and HR mock interview simulations
* 🧠 LLM-powered answer evaluation and feedback
* 📊 Performance scoring and interview analytics
* 🎤 Voice-based interview support
* 📝 Interview history and progress tracking
* 🔐 Secure user authentication
* 📱 Responsive and modern user interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/amajali-784/AI-Interview-Coach.git
cd AI-Interview-Coach
```

### Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt
```

Create a `.env` file inside the `backend` directory:

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

Start the backend server:

```bash
python app.py
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
